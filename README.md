# Rover3DFiles
Rover 3D Printed files
<b>Turkish:</b> Bu bölümde projenin bir parçası olan, İnternet üzerinden kontrol edilecek ve üzerinde çeşitli görev yükü bulunacak olan model Mars Rover'a ait 3D yazıcı dosyaları yer almaktadır.
<b>Malzeme:</b> Malzeme olarak istenilen renkte filament kullanılabilir. Çalışmalarda kullanılan filament türü PLA'dır. Dosyalar 180-220 Nozzle, 55-60 tabla sıcaklığında, %7 doluluk oranında basılabilir.
